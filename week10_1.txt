1교시
git swtich -c 브랜치명 =git ckeckout -b 브랜치명 (브랜치 생성 후 바로 브랜치 전환)
ckeckout의 기능이 너무 많아 switch 등의 명령어로 분리됨
git branch -d 브랜치명 (브랜치 삭제, d=delete)
브랜치 삭제해도 다른브랜치 파일은 남아있음
rm ai_0* (ai_01, ai_02 등의 파일 전체 지움)
git restore 삭제 파일 복구
rm -rf .git (.git 파일 강제 삭제, -rf 강제 삭제명령어)
git push origin --delete (브랜치 삭제)

2교시 
vi에디터 사용
vi week101.txt (vi 에디터 일반모드 )
i 누르면 입력모드
esc 누르면 일반모드로 전환
:q! vi 에디터에서 나오기

일반모드에서
x 누르면 글씨 하나 삭제
dd 현재 커서 있는 행 삭제
p 붙여넣기
yy 복사
esc :wq! 저장 후 나오기

입력모드에서는 글 수정 가능
